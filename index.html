Phage Therapy Simulation-Discrete Events

A discrete event model that simulates interactions between co-inhabiting populations of bacteriophage lambda and bacteria.

Download this project as a .zip file Download this project as a tar.gz file

Description

This is a discrete event model that has been developed to computationally simulate the many interactions that occur between co-inhabiting populations of bacteriophage lambda and bacteria.
